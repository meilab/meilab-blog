C语言中有时候多一个或者少一个分号会导致程序出现很低级的错误。

注意下面代码中if判断之后，如果多了一个分号，就会导致程序出现隐晦的bug

```c
int max(int a, int b)
{
    int ret = a;
    if (a < b);
        ret = b;
    return ret;
}

main()
{
    int ret = max(30, 20);
    printf("%d", ret);
}
```

需要使用现代化的IDE，保存后能够帮我们优化代码展示格式，从而发现潜在的问题。
